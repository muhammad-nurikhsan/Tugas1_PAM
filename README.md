## Deskripsi Tugas

Project ini merupakan aplikasi sederhana berbasis **Kotlin Multiplatform (Compose Multiplatform)**.  
Aplikasi dibuat sebagai tugas praktikum untuk menampilkan informasi dasar mahasiswa berupa:

- Nama
- NIM
- Platform tempat aplikasi dijalankan

Program ini berfungsi sebagai implementasi awal konsep pengembangan aplikasi mobile lintas platform menggunakan Kotlin dan Jetpack Compose.

---

## Fitur Aplikasi

- Menampilkan identitas mahasiswa
- Tampilan UI sederhana berbasis Compose
- Dapat berjalan pada platform Android
- Struktur project menggunakan Kotlin Multiplatform

---

## Environment

- **JDK:** 17  
- **IDE:** Android Studio  
- **Framework:** Compose Multiplatform  
- **Bahasa Pemrograman:** Kotlin
